This project is a fully autonomous six-legged walking robot — a **hexapod** — designed and
built from the ground up. The robot integrates custom mechanical linkages, embedded electronics,
and real-time gait-control algorithms to achieve stable locomotion across a variety of surfaces.

The hexapod is controlled by an onboard microcontroller that coordinates eighteen degrees of
freedom (three servo-actuated joints per leg) in concert, producing smooth, bio-inspired
movement patterns. An optional front-mounted camera module allows the operator to steer the
robot remotely while receiving a live video feed.

Key features of the project include:

- **Custom CAD-designed frame** — every structural component was modelled in CAD and 3-D printed.
- **Inverse kinematics engine** — analytical IK solves each leg's joint angles in real time.
- **Tripod gait implementation** — alternating sets of three legs keep the robot statically stable at all times.
- **Simulation-first workflow** — the gait was prototyped and validated in CoppeliaSim before any hardware was assembled.
- **Wireless control** — an onboard antenna module provides low-latency telemetry and command reception.
